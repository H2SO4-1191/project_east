# Project East – Multi-User Education Web Application

Project East is a large-scale web platform connecting students, lecturers, and institutions in a social-media-style ecosystem. Built with Django and PostgreSQL, it supports course enrollment, job postings, messaging, payments, and AI-powered document verification.

## Features

- Multi-role support: students, lecturers, institution managers  
- 80+ API endpoints for targeted feeds, courses, and job opportunities  
- AI model for document classification  
- Admin dashboards for managing users, courses, payments, and subscriptions  
- Automated scheduling, attendance, and grade tracking  
- Payment integration using Stripe  
- Deployed with Docker on Google Cloud  

## Tech Stack

- Backend: Python, Django, PostgreSQL  
- Frontend: HTML, CSS, JavaScript  
- AI: Python, Ultralytics YOLO (document classifier)  
- Deployment: Docker, Google Cloud  

## Setup & Usage

- python -m venv venv
- source venv/bin/activate  # Linux/Mac
- venv\Scripts\activate     # Windows
- pip install -r requirements.txt
- python manage.py migrate
